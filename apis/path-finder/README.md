# Path Finder API ![license](https://img.shields.io/badge/License-LGPL-green.svg?label=license)

## Usage

### V0

tokenId of NativeToken is 'Native' or address '0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE'

Types

```ts
interface QueryParams {
  chainId: number
  fromTokenId: string
  toTokenId: string
  gasPrice: number
  amount: string
  to?: string
  priceImpact?: number
}
```

Example

```ts
// DEFAULT_API_URL: https://path-finder.zenlink.pro
const res = await (
  await fetch(`
    ${API_URL}/v0?chainId=${
      chainId
    }&fromTokenId=${
      Native | address
    }&toTokenId=${
      Native | address
    }&amount=${
      amount
    }&gasPrice=${
      gasPrice
    }&priceImpact=${
      slippageTolerance / 100
    }${recipient ? `&to=${recipient}` : ''}`
  ).json()
)
```
